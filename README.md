# OmniServe

Unified AI Engineer Portfolio System – a local‑first, real‑time, multimodal AI assistant that answers questions from documents, images, and live audio/video streams.

## Features (coming soon)
- Chat + Document QA with citations
- Image understanding via OCR + object detection
- Live voice assistant (ASR + TTS)
- Real‑time log analysis and anomaly detection
- Admin dashboard with full observability
- CI/CD regression testing

Built using FastAPI, Streamlit, Ollama, Qdrant, Langfuse, and more.
